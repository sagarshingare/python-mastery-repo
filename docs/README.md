# Documentation Hub

This folder contains architecture documentation, interview notes, cheatsheets, diagrams, and best practices for mastering Python engineering.

## Structure

- `architecture/` - high-level design patterns, system diagrams, service decomposition examples.
- `interview_notes/` - curated interview questions, answers, and behavioral frameworks.
- `cheatsheets/` - quick reference guides for Python, data engineering, and DevOps.
- `diagrams/` - architecture diagrams and flowcharts.
- `best_practices/` - production-grade design and implementation patterns.
